HOW TO PLAY (操作方法)

INVADERS

  • ← → : 左右移動
  • Space : ミサイル発射
  • ・トーチカに隠れて弾を防ごう
  • ・時折出現するUFOは高得点！

TETRIS

  • ← → : 左右移動
  • ↑ : ブロック回転
  • ↓ : ソフトドロップ
  • Space : ハードドロップ
  • ・4ライン同時に消して「テトリス」を狙おう

HEAD-ON

  • Space / Click : 車線変更
  • ・敵の車と正面衝突しないよう注意！
  • ・コース上のすべての黄色いドットを回収しよう
  • ・スピードアップボタンで敵を引き離せ

PACMAN

  • WASD / ↑↓←→ : キーボード移動
  • Swipe / D-Pad : スマホ操作対応
  • ・パワーエサを食べるとゴーストを撃退可能！
  • ・すべてのクッキーを食べ尽くせ！

MAHJONG

  • Click : 牌を選択して打牌
  • ・チー・ポン・カン・ロン・ツモは画面にボタン表示
  • ・ルームIDで遠くの友人とリアルタイム通信対局
  • ・人数不足時はCOM（AI）が自動補填されます

GOMOKU

  • Click : 盤面をタップして石を置く
  • ・黒か白の石を一直線に5つ並べたら勝利！
  • ・ルームIDで友達とオンライン通信対戦が可能
  • ・オフラインのCOM（AI）練習モードも搭載

REVERSI

  • Click : 盤面をタップして石を置く
  • ・自分の色の石で相手の石を挟むと反転します
  • ・最終的に盤面で自分の石が多い方が勝利！
  • ・オンライン通信対戦＆COM対戦に対応

BLOKUS

  • Click : ピースを選択して盤面へクリック配置
  • R / Space : ピースを回転
  • F / X : ピースを反転
  • ・自分の色のピースと角のみが接するように置こう！
  • ・オンライン通信対戦＆COM対戦に対応

SHOGI

  • Click / Drag : 駒を選択して移動・打つ
  • ・成れる移動時には画面に成り・不成ボタンが表示されます
  • ・ルームIDで遠くの友人とリアルタイム通信対局が可能
  • ・難易度3段階のCOM対戦（Web Worker AI）を搭載

MILITARY SHOGI

  • Click / Tap : 駒を選択して配置・移動
  • ・戦闘時は審判が勝敗を自動判定（正体は非公開）
  • ・ルームIDで友達と非対称情報の通信対局が可能
  • ・不完全情報下の推測メモリを搭載したCOM（AI）対戦
